# meowpixel (v0.1 beta)
### meowpixel is a very simple pixel art maker program. Free-to-use, open source and fast! Enjoy! (:

---

## Features
- **Simple**: meowpixel is a *really* **simple** program to learn.
- **Fast**: meowpixel is *really* **fast and powerful** program. Draw your pixel arts with no fuss!
- **Free-to-use**: meowpixel is **completely** free for *everyone*.
- **Open Source**: meowpixel is **open source** so you can change source code **anytime**!

## Built With
- **C** (Programming Language)
- **raylib** (Game Library)

---

## About
meowpixel was created in **C** using **raylib** by **Ice748** as a pet-project in May-June of 2026. I made it with a *broken keyboard* — keys **2**, **W**, **S**, **X** and **Caps** *don't work*. I wrote them using *virtual keyboard* and this caused me a lot of *difficulties*, so, please, **support** this project! ❤️
